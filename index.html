Data Collection

This app collects the following data solely for functionality purposes:

All credentials are stored locally on your device using Android's built-in SharedPreferences. No data is transmitted to third parties or external servers except the server URL you configure.

We do not sell, trade, or share your personal data with third parties. Your data is only used to provide the core functionality of the app.

Security

All network communication with your configured server uses HTTPS encryption where available. Device identifiers are hashed, not stored in plain text.
